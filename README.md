# superPerformanceScreener
SuperPerformanceScreener is an AI-powered stock screening tool that automatically detects explosive and superperformance growth patterns across NYSE and NASDAQ stocks based on customizable time and return thresholds. It integrates real-time market data (via MCP APIs) and uses Perplexity's LLM to generate concise, research-driven summaries explaining the drivers behind each flagged stock surge. Results are auto-logged to Google Sheets, making this a streamlined system for discovering high-growth equities with full transparency.
